📱 아이패드 / 아이폰에서 SSM 실행 방법

모바일 Safari/Chrome은 로컬 파일에서 보안상 스크립트 실행 및 저장 공간을 제한합니다. 아이패드/아이폰에서 터치가 원활하게 동작하고 동기화되게 하려면 아래 방법 중 하나를 선택하세요.

방법 1: GitHub Pages 무료 웹페이지로 사용 (강력 추천)
  1. 이 프로젝트가 올라와 있는 깃허브 홈페이지(GitHub Repository)에 접속합니다.
  2. [Settings] -> [Pages] 탭으로 이동합니다.
  3. Build and deployment에서 [Source]를 "Deploy from a branch"로 선택하고, Branch를 "main" (또는 "master"), 폴더를 "/ (root)"로 설정한 뒤 [Save]를 누릅니다.
  4. 약 1~2분 후 생성되는 공개 주소(예: https://Ar9av.github.io/obsidian-wiki/Saint_Serving_Master/index.html)로 기기에서 접속하시면 홈화면에 바로가기 앱처럼 100% 정상 가동됩니다!
방법 2: 아이패드 내부 로컬 웹서버 구동 (오프라인용)
  1. 아이패드 앱스토어에서 무료 HTML 에디터/서버 앱인 **Koder** 또는 **Textastic**을 설치합니다.
  2. 이 Saint_Serving_Master.html 파일을 해당 앱으로 가져옵니다.
  3. 앱 내의 [Play/Run] 버튼을 누르면 내부 호스트 주소(http://127.0.0.1:8080)가 가동되어 온전히 실행됩니다.
